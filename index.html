About 私のこと

田中 瑞樹 - Mizuki Tanaka

1999年福井県生まれ。 小さい頃からものづくりが好きで、学生時代までは家業である「提灯の絵付け」を手伝っていました。 その後デザインに興味を持ち、アルバイトで漫画サイトのバナー広告を制作。 アルバイトをしていく中で、もっと動きのあるWebサイトに魅力を感じ、現在6か月間のスクールに通い勉強中。

Skills できること

Design

Webサイトやバナーなどのデザイン制作をしています。
細部まで丁寧に作りこむことを意識し、機能性や情報の分かりやすさにも配慮しています。
ただ見た目が良いだけでなく、使う人にとって役立ち、印象に残るようなデザインを目指しています。

Tools : Illustrator / Photoshop / Figma

Coding

また、保守性と可読性を意識した、ルールに沿ったコーディングを心がけています。

Tools：Visual Studio Code
